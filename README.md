شرح المكتبات:
react-hook-form و zod: لإنشاء نماذج قوية والتحقق من صحة البيانات بسهولة (احترافي جدًا).
resend: الخدمة التي سترسل البريد الإلكتروني.
react-hot-toast: لإظهار إشعارات جميلة للمستخدم (مثل "تم الإرسال بنجاح").
